Projects

Headcount

Headcount is relational database built on a collection of Colorado state education data from Data.gov. Using object-oriented programming, we answered questions about the relationships between median household income, kindergarten participation, high school graduation, and statewide test scores.

Battleship

This is a text-based, terminal-run implementation of the classic game Battleship. A user plays against an AI by placing ships on a board and firing at the AI's board.

CompleteMe

CompleteMe is an auto complete application. The backbone of Complete-Me is a "trie" tree, short for a re-trie-val tree. The trie is used for storing and fetching paths that map out possible words. Complete-Me is a broadly-branched tree where paths from the root to the leaves represent “words” within the dictionary.
